# Landing-page queue — keyword-targeted (5/day)

Programmatic SEO for crypto-license consulting. Each page targets a specific
high-intent keyword. Daily pipeline takes the next 5 unchecked `- [ ]`, builds a
~1800-2000 word landing page at the given slug, links it into the jurisdictions
hub + sitemap + IndexNow, marks it `- [x]`.

Page = commercial landing page (not blog): keyword in <title>/H1/meta, real
jurisdiction facts (regulator, license type, cost band, timeline, capital, tax,
who it suits), comparison table, FAQ schema, and a Consulting24 consultation CTA
(WhatsApp). Internal-link 3-5 related jurisdiction/variant pages. Honest: where
Consulting24 places via partners, say "we advise + coordinate", not "we file".

Primary keyword template per slug: "<Jurisdiction> Crypto License <year>: Cost,
Requirements & How to Get One".
